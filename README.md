# Angular HR Dashboard

## 📌 Project Overview
This is a modern HR Management Dashboard built using Angular.  
It includes analytics, attendance tracking, and task management UI.

---

## 🚀 Features
- Sidebar Navigation
- Dashboard Analytics Cards
- Attendance Heatmap
- Task Management (Kanban Board)
- Responsive UI Design
- Clean and Modern Layout

---

## 🛠️ Tech Stack
- Angular
- TypeScript
- HTML
- CSS
- Lucide Icons
